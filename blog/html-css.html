Box Model Basics

The box model is a CSS concept which allows the browser to view all created content is contained within a box. Each box has three main properties, they are:

Margin

The margin is the space outside of the border of an element, and is between other elements. It is usually displayed as orange in colour when you inspect an html page. Margin properties are:

  • margin-left
  • margin-right
  • margin-top
  • margin-bottom
